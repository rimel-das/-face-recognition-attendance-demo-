# Contributing to Face Recognition Attendance System

We'd love your contribution! Here's how to get started:

## Getting Started

1. **Fork the repository** on GitHub
2. **Clone your fork** locally:

   ```powershell
   git clone https://github.com/YOUR-USERNAME/face-recognition-attendance.git
   cd face-recognition-attendance
   ```

3. **Create a feature branch**:

   ```powershell
   git checkout -b feature/your-feature-name
   ```

4. **Install dependencies**:
   ```powershell
   pip install -r requirements.txt
   ```

## Making Changes

- Write clean, readable Python code
- Follow PEP 8 style guidelines
- Add comments for complex logic
- Test your changes locally with `python app.py`

## Submitting Changes

1. **Commit your changes**:

   ```powershell
   git add .
   git commit -m "Brief description of changes"
   ```

2. **Push to your fork**:

   ```powershell
   git push origin feature/your-feature-name
   ```

3. **Create a Pull Request** on GitHub with:
   - Clear title describing the change
   - Description of what was changed and why
   - Link to any related issues

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Help others learn and grow

## Questions?

Feel free to open an issue or reach out. We're here to help!
